LENOVO-IDEAPAD-720S-13IKB-I5-8250U
========

系统配置
---
硬件| 型号
----|----
CPU | I5 8250U
RAM | 8G DDR4 2400MHz
DISK |三星 PM961
Display| LG FHD
WIFI |BCM94352z（DW1560）
<br> 

安装注意事项
----
关闭Intel SGX<br> 
关闭Secure BOOT<br> 
不要使用BCM94350(FRU00JT494),极其卡顿，无法使用<br> 
GPIO int注入0x1B 否则与VoodooI2C不匹配，导致无法降频<br> 

存在的问题
----
触摸板偶尔失效<br>
